# Spotlight Dark Mode Toggle

A cyberpunk-inspired dark mode toggle featuring a spotlight sweep animation, glowing neon effects, and a smooth sliding thumb. Built entirely with HTML and CSS, it is lightweight, responsive, and requires no JavaScript.

## Features

* Pure HTML & CSS
* Spotlight sweep animation
* Neon cyberpunk glow
* Smooth toggle transition
* Responsive design
* Keyboard accessible

## Usage

Include `style.css` in your project and use the HTML structure from `demo.html` to add the toggle to your page.

## Why EaseMotion CSS?

This component follows EaseMotion CSS's animation-first philosophy by providing a reusable, visually appealing UI element with lightweight CSS-only animations. It is suitable for dashboards, landing pages, settings panels, and modern web interfaces.
